# 📌 Data Sources

## 1. Global Events
- **GDELT**: Structured CSV/JSON feed updated every 15 minutes
  - Covers: war, conflict, political events, protests, disasters
  - Notes: Use CAMEO codes to filter relevant events

## 2. Commodity-Specific Events
- **NewsAPI**: JSON API, hourly fetch
  - Keywords: opec, oil, energy, natural gas, gold, federal reserve, central bank, market


## 3. Commodity Prices
- **Alpha Vantage API**: Intraday or hourly prices for oil, gas
  - Notes: Structured JSON; include symbol, timestamp, price
- **Yahoo Finance / yfinance**: Prices for gold, silver, other commodities
